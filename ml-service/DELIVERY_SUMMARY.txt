QUICK START
================================================================================

1. Install dependencies:
   pip install -r requirements.txt

2. Train model:
   python3 train.py

3. Run API:
   python3 -m uvicorn api.main:app --host 0.0.0.0 --port 8000

4. Run tests:
   python3 test_integration.py
